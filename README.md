# Todo Calendar

A smart task planner that schedules tasks using rule-based scoring.

## Features
- Add and manage tasks
- Automatic daily scheduling
- Priority-based scoring system
- Clean UI (sky blue + burgundy theme)

## Tech Stack
- React / Next.js
- TypeScript
- Vercel (deployment)

## Live Demo
https://todo-calendar-project-7lknulvud-sgaidams-projects.vercel.app

## How it works
Tasks are ranked based on importance, deadlines, and constraints, then scheduled into a daily plan.
